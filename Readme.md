### Technologies Used
## Frontend:
React.js 
Tailwind CSS 
## Backend:
Node.js & Express.js (REST API)
MongoDB & Mongoose (Database)
JSON Web Token (JWT) for authentication

### Installation & Setup
## Install Dependencies
# Backend
cd backend
npm install

# Frontend
cd Frontend
npm install

###  Running the Project
## Start Backend Server
cd backend
npm  Start

### Start Frontend Server
cd frontend
npm start
